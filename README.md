# BitwigPlaybackUtility

Bitwig Studio用のコントローラースクリプトです。再生開始時に指定した拍数のカウントインを行い、マスターボリュームをフェードインします。

## 機能

- 再生ボタンを押すと同時にマスターボリュームを指定ボリュームに設定
- メトロノームで指定拍数カウントイン（4 / 8 / 16 / 32 拍から選択）
- カウントイン終了直前（指定ms）にマスターボリュームをフェードアップ
- カウントイン完了後、メトロノームを自動でオフ
- ツールバーポップアップからカウントイン機能のON/OFF・拍数・ボリューム・フェード時間を設定可能
- カウントインON時、停止中に再生位置を変更すると自動でカウントイン拍数分だけ手前にオフセット

## インストール

1. このフォルダを Bitwig Studio のコントローラースクリプトディレクトリに配置します
   - macOS: `~/Documents/Bitwig Studio/Controller Scripts/`
2. Bitwig Studio を起動し、**Settings > Controllers** を開きます
3. `Add controller` から `Custom > BitwigPlaybackUtility` を選択します

## 使い方

インストール後、再生ボタンを押すだけで自動的にカウントインが始まります。

### カウントインのON/OFF

ツールバーのコントローラーアイコンをクリックしてポップアップを開き、**Count-in** を `ON` / `OFF` で切り替えます。

- **ON → OFF 切り替え時**: メトロノームがONの場合は自動でOFFになります
- **OFF → ON 切り替え時**: 停止中であればメトロノームが自動でONになります（再生中は変更しません）

### カウントイン拍数の変更

同じポップアップの **Count-in Beats** で `4` / `8` / `16` / `32` 拍を選択できます（デフォルト: 8）。

### カウントイン中のボリューム

**Count-in Volume** でカウントイン中のマスターボリュームを設定できます（デフォルト: -24dB）。

| 選択肢 | 説明 |
|--------|------|
| 0dB | 通常音量 |
| -18dB | |
| -24dB | デフォルト |
| -32dB | |
| -∞dB | 無音 |

### フェード時間の設定

**Fade Duration** でカウントイン終了前のフェードアップ時間をms単位で入力できます（範囲: 10〜2000ms、デフォルト: 100ms）。テンポに関わらず常に指定した時間でフェードします。

### 再生位置のオフセット

カウントインがONの場合、停止中に再生位置を変更すると、設定した拍数分だけ自動で手前にオフセットされます。これにより再生ボタンを押したとき、カウントイン終了後にちょうど指定した位置から演奏が始まります。

## 動作環境

- Bitwig Studio 5.x
- Controller API 19
